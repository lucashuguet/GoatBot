## Getting Started
### Prerequisites
-   [Node.js](https://nodejs.org/) >= 12.16.3
-   npm >= 6.14.5
### Installation
``` bash
# Clone the repository (master branch)
git clone https://github.com/lucashuguet/GoatBot.git

# Go to the project root
cd GoatBot

# Install
npm install

# Add your token
# on linux
export token=<your_token>
# on windows
set token=<your_token>
```
### Usage
``` bash
node index.js
```
